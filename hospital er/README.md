## Overview
This project is an Excel-based report that analyzes monthly emergency room (ER) data.  
It helps hospitals track patient visits, case types, and demographic details for better decision-making.

## Objectives
- Count total ER visits each month.  
- Show age group and gender distribution of patients.  
- Highlight the most common emergency types.  
- Detect seasonal or peak-time patterns.  
- Provide a dashboard for quick insights.  

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts (Line, Column, Pie)  
- Conditional Formatting  
- Slicers & Filters

## How to Use
1. Open **Hospital_ER_Report.xlsx**.  
2. Go through these sheets:  
   - **Raw Data** ‚Üí Patient records  
   - **Pivot Analysis** ‚Üí Summaries  
   - **Dashboard** ‚Üí Visual insights  
3. Use slicers to filter by month, gender, or emergency type.  

## Key Metrics
- Monthly patient visits  
- Emergency type breakdown  
- Age and gender analysis  
- Peak months and trends  

## Insights
- Shows surge patterns in ER visits.  
- Helps allocate staff and resources.  
- Provides demographic insights for better planning.
